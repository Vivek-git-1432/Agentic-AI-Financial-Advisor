# 💰 Agentic AI Financial Advisor

An intelligent AI-powered financial management web application built using Streamlit, OCR, and Groq LLM.

This project helps users track expenses automatically from payment screenshots, analyze spending behavior, and receive personalized AI-based financial guidance.

---

# 🚀 Features

## 📷 OCR-Based Expense Detection
- Upload payment screenshots
- Extract expense information using Tesseract OCR
- Automatically detect amount and expense category

## 🤖 Agentic AI Financial Advisor
- AI-powered financial assistant using Groq LLM
- Personalized financial advice
- Intelligent financial reasoning
- Investment and savings suggestions
- Financial risk analysis

## 📊 Dashboard Analytics
- Expense tracking dashboard
- Budget utilization analysis
- Expense distribution charts
- Monthly expense trends
- AI smart alerts

## 🧠 Personalized AI Memory
- AI remembers user financial profile
- Personalized financial recommendations
- Conversation memory support

## 📝 Expense Management
- Automatic expense entry
- Manual expense entry
- Edit expenses
- Delete expenses
- Expense history tracking

## 📑 AI Financial Reports
- Generate complete AI-based financial reports
- Spending behavior analysis
- Savings suggestions
- Budget improvement recommendations

# ⚙ Installation

Install Dependencies
pip install -r requirements.txt


Run Application
streamlit run app.py
🔑 API Setup


Get your Groq API Key from:
https://console.groq.com/keys


Enter the API key in the sidebar while running the application.
